# rx-pushservices-except-mipush
针对除小米服务外国内各类推送服务的屏蔽处方

[MiPushFramework](https://github.com/Trumeet/MiPushFramework)/小米用户可用

-更新时间：2018/01/31

仅支持Greenify 3.3+

请注意：本处方已使用Greenify v3格式编写，但仍然无法保证所有处方条目可正常工作，导入该处方请自行承担后果！
